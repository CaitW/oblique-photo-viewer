    public legendStyleUpdate(layerId: string, propertyName: string, style: Object, geometryType: string): {"type": string, "layerId": *, "propertyName": *, "style": *, "geometryType": *} source 

    import {legendStyleUpdate} from 'oblique-photo-viewer/src/js/ducks/layers.js'

    When the app is processing a layer and comes across a new feature style, it dispatches this action to store the style in the store. It's then used by the legend to display an appropriate entry.

    Params:

    NameTypeAttributeDescription
    layerId string
    propertyName string

    the name of the property within the feature that's being styled i.e. in the Structures layer, we're styling different structure types
    style Object

    the key/value pairs that describe a Leaflet style
    geometryType string

    usually point or line, allows the legend to determine what shape to show as an icon next to a legend entry

    Return:

    {"type": string, "layerId": *, "propertyName": *, "style": *, "geometryType": *}
