    public openMobileLayerList(): {"type": string} source 

    import {openMobileLayerList} from 'oblique-photo-viewer/src/js/ducks/mobile.js'

    mobile.js Ducks

    Contains the actions and reducer part that controls mobile functionality. Mobile functionality is present when the window is smaller than 992px. These features include:

    • Mobile Layer List
    • Keeping track of window size
    • When a feature is clicked during mobile mode (<992px)
    • Mobile Feature Modal

    {"type": string}
